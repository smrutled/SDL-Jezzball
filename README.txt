Game: Jezzball clone

Visual Studio 2013
Dev Library: SDL 2.0.1

The libraries are included with in the project.
Open project and run.

Click on Jezzball title to play
Press p to pause.
You get 5 lives.
Has infinite levels
Returns to title when lives are up
